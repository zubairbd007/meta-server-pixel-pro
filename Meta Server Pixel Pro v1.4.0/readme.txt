Advanced Meta Pixel & Conversions API plugin with server-side tracking and premium UI.

== Description ==

Meta Server Pixel Pro is a professional WordPress plugin that allows you to send Meta (Facebook) Pixel events using both browser and server-side (Conversions API) tracking.

This helps improve tracking accuracy, reduce data loss from ad blockers, and improve Meta Ads optimization.

**Key Features:**
- Server-side Meta Conversions API (CAPI)
- Browser + Server tracking support
- Event-wise toggle control
- WooCommerce event tracking
- Event deduplication support
- Premium, clean admin UI
- Lightweight and production-ready

== Installation ==

1. Download the plugin ZIP.
2. Go to WordPress Admin → Plugins → Add New → Upload Plugin.
3. Upload the ZIP file and activate.
4. Go to **Meta Server Pixel Pro** from admin menu.
5. Enter your Meta Pixel ID and Access Token.
6. Enable events and save settings.

== Usage ==

- Enable Browser Pixel, Server Pixel, or both.
- Select which events you want to track.
- For WooCommerce sites, product, cart, checkout and purchase events are tracked automatically.
- Use Test Events in Meta Events Manager to verify server-side events.

== Frequently Asked Questions ==

= Does this support WooCommerce? =
Yes. WooCommerce events like ViewContent, AddToCart, InitiateCheckout, and Purchase are supported.

= Is server-side tracking enabled by default? =
You can enable or disable browser and server tracking independently from the plugin settings.

= Is this an official Meta plugin? =
No. This is an independent plugin built using Meta’s public Conversions API.
